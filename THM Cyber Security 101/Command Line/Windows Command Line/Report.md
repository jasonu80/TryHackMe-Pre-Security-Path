
# Command Line - Windows Command Line - TryHackMe Walkthrough

[Room Link](https://tryhackme.com/room/windowscommandline)

## Hands-on lab description and impact

This lab provides the ability for the user to be able to use the Windows Command Line interface, which Windows has its application called `cmd.exe`. There are several tasks that this lab has done, such as viewing basic system configuration, performing a network configuration and troubleshooting, viewing and making as well as deleting files and directories, and view the process of task manager. This provides a basic overview on how to use the command line and enhances our productivity to reduce the number of clicks that are not needed. 

#### Impact

It will increase the productivity of the user and obtain the ability to read syntax and documentation, similar like the linux command line.
