
# Command Line - Linux Shells - Summary

[Room Link](https://tryhackme.com/room/linuxshells)

## Hands-on lab description and impact

The room discusses about how to interact with the Linux shell, revise the UNIX commands that are discussed in Linux Fundamentals module, explore different types of shells, and write a shell script. The objective that has the most impact is the ability to create a shell script to find a number of vulnerabilities in a system or solve a particular problem that the industry needs, such as the application of regex in the shell. The hands-on lab allows the user to learn commands that are listed on the room, such as `history` that retrieves the list of commands ran in a system.

It is crucial that a learner understands the basic language of shell scripting as it is a language that most Cyber Security professionals will utilise in the industry.
